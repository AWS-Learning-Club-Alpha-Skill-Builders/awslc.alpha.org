## ◼ Creative Development

### Animation & Motion Design
- **GSAP (GreenSock)** — Advanced timelines, ScrollTrigger, SplitText, MorphSVG, DrawSVG, Flip plugin. Orchestrating complex, multi-layered scroll-driven narratives.
- **Framer Motion** — Layout animations, shared element transitions, spring physics, gesture-driven interfaces in React ecosystems.
- **Lenis / Locomotive Scroll** — Butter-smooth custom scroll experiences with parallax layering, velocity-mapped effects, and scroll-hijack sequences.
- **CSS Animations & Transitions** — Hardware-accelerated keyframes, custom easing with `cubic-bezier`, `@scroll-timeline`, View Transitions API.
- **FLIP Technique** — First-Last-Invert-Play pattern for performant, layout-aware animations at 60fps.
- **Anime.js / Motion One** — Lightweight animation orchestration for micro-interactions and UI state transitions.

### WebGL & 3D
- **Three.js** — Custom shaders, post-processing passes (Bloom, Chromatic Aberration, Film Grain), instanced meshes, raycasting, particle systems.
- **React Three Fiber (R3F)** — Declarative 3D scenes in React with Drei helpers, physics (Rapier/Cannon), and performant reconciliation.
- **GLSL / Shader Programming** — Vertex & fragment shaders, noise functions (Simplex, Perlin, Worley), signed distance fields, raymarching, framebuffer feedback loops.
- **OGL** — Lightweight WebGL library for bespoke creative projects requiring fine-grained control.
- **Spline / Rive** — Interactive 3D and motion graphics embedded seamlessly in production builds.

### Canvas & Generative Art

- **p5.js / Processing** — Generative and algorithmic art, interactive installations.
